开始运行

开始运行

Build

运行bin/build.sh来构建docker镜像rails-devise-jw。

Run

创建镜像后，运行：docker-compose up -d 然后，用docker compose exec server rails db:create初始化数据库。

Test

rails db:reset db:seed RAILS_ENV="test"
rspec

CORS

在默认情况下，Rails-pangs允许CORS并公开Authorization的Header。如果你想要禁用它，可以注释掉config/cors.rb文件的内容。
